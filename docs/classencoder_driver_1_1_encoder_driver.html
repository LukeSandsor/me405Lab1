◆ update()

def encoderDriver.EncoderDriver.update (   self)

Handles overflow for the counter by creating a differential between the last counted number and the counted number at run time.

If there is overflow, the differential will be greater or less than the period (-self.period/2 to self.period/2 is equals to one period) and the code will adjust the number accordingly
